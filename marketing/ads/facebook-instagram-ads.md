## Ad Creative Best Practices

### Image Specifications
- Feed: 1080 x 1080px (1:1)
- Stories/Reels: 1080 x 1920px (9:16)
- File type: JPG or PNG
- Max file size: 30MB

### Video Specifications
- Feed: 1:1 or 4:5
- Stories/Reels: 9:16
- Length: 15-60 seconds (15 sec for Stories)
- Captions: Always include (85% watch without sound)

### Copy Guidelines
- Primary text: 125 characters optimal (truncates on mobile)
- Headline: 40 characters max
- Use emojis sparingly for visual break
- Include social proof where possible
- Clear, single CTA

### Creative Testing Framework
- Test 3-5 creatives per ad set
- Rotate every 2-4 weeks to prevent fatigue
- A/B test: Images vs. Video, Testimonials vs. Features
- Monitor frequency - pause when >3

## Audience Targeting Guide

### Core Audiences
1. **Interest-Based:** Dating, Relationships, Love, Romance, Marriage
2. **Behavior-Based:** Engaged shoppers, Recently single, Active daters
3. **Demographic:** Ages 25-45, All genders, Urban/Suburban

### Lookalike Audiences
1. App installers (1%, 3%, 5%)
2. Registered users (1%, 3%)
3. Premium subscribers (1%)
4. High-engagement users (1%)

### Retargeting Audiences
1. Website visitors (180 days)
2. App installers (no registration)
3. Registered users (inactive 7-30 days)
4. Free users (active, for premium upsell)

### Exclusions
- Current premium subscribers
- Recent converters (7 days)
- Employees/team members
